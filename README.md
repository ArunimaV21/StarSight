# StarSight
An Urban Astronomy Observation Analysis Tool
## Current Version
StarSight V5
## Project Status
Active Development
## Overview
StarSight is a Python-based astronomy observation analysis tool designed to evaluate observing conditions and track observation quality over time.
The project records environmental conditions such as cloud cover and moon brightness, calculates an observation score, stores observations in a CSV database, and generates statistical analyses and visualizations.
## Features

### Observation Recording

- Date recording
- Cloud cover recording
- Moon brightness recording
- Automatic night numbering

### Analysis

- Observation score calculation
- Observation quality classification
- Observation recommendations

### Statistics

- Best observation night
- Worst observation night
- Cloudiest night
- Brightest moon night
- Average observation score
- Observation quality distribution
- Most common observation quality
- Best environmental conditions summary

### Data Visualization

- Line graphs
- Bar graphs
- Histograms
- Comparison graphs
- Graph export
## Technologies Used

- Python
- Pandas
- Matplotlib
## Installation

Install required libraries:

pip install pandas matplotlib

Run the program:

python starsight.py
